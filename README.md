# Electric Menu Backend

Backend API untuk aplikasi Electric Menu menggunakan Express dan PostgreSQL.

---

## Features

- Authentication (JWT)
- Manage Shops
- Manage Products
- Product Types
- QR Code for shop

---

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- JWT
